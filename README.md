## 🎯 מה זה?

**מגמת צמיחה** היא אפליקציית PWA (Progressive Web App) ללימוד שוק ההון הישראלי, מיועדת לתלמידים ומשקיעים מתחילים.

האפליקציה מאפשרת לעקוב אחרי מניות ת"א 35 בזמן אמת, לנהל תיק דמה וירטואלי, ללמוד מושגים פיננסיים, ולתרגל החלטות השקעה — הכל בסביבה בטוחה ללא כסף אמיתי.

---

## ✨ פיצ'רים עיקריים

### 📈 מעקב מניות
- נתוני 35 מניות ת"א 35 בזמן אמת
- גרפים אינטראקטיביים (יום / שבוע / חודש / 3 חודשים / שנה)
- דף מניה מפורט עם נתונים פיננסיים
- סינון לפי עולים/יורדים/שווי שוק/נפח
- חיפוש חופשי

### 💼 תיק דמה
- ₪250,000 כסף וירטואלי להשקעה
- קנייה לפי כמות או לפי סכום (₪)
- מעקב רווח/הפסד בזמן אמת
- גרף פיזור תיק לפי ענפים
- יומן עסקאות מפורט

### 🎮 Trader Academy
- 15 תרחישי השקעה אינטראקטיביים
- 3 רמות קושי (מתחיל / בינוני / מתקדם)
- כל סיבוב: אירוע → החלטה → לקח
- נושאים: ריבית, אינפלציה, פאניקה, FOMO, מידע פנים, DCA ועוד

### 📖 מילון מושגים
- 26 מושגים פיננסיים
- 5 קטגוריות (יסוד / אסטרטגיה / מדדים / שוק / סיכון)
- חיפוש חי וסינון לפי קטגוריה

### 🏆 תחרות משקיעים
- דירוג אוטומטי של כל המשתמשים
- צפייה בתיק של חברים
- חיפוש חברים לפי שם משתמש

### ⚖️ כלי השוואת מניות
- בחירת 2 מניות להשוואה
- טבלת השוואה (מחיר, שווי שוק, נפח, שינוי)
- גרף ביצועים השוואתי

### 🎯 אתגרים ומשימות
- 15 אתגרים ב-3 רמות
- מעקב התקדמות אוטומטי
- לחיצה על אתגר → ניווט ישיר לביצוע

### 💰 ניהול תקציב
- מחשבון תקציב חודשי
- 4 קטגוריות הוצאות
- חישוב "כסף פנוי להשקעה"

## 🚀 התקנה

### שימוש מיידי
גש ל: **https://ethanmimoun.github.io/ta35-stocks/**

### התקנה כ-PWA
1. פתח את הקישור בכרום
2. לחץ "הוסף למסך הבית" (Add to Home Screen)
3. האפליקציה תופיע כאייקון בטלפון

### פיתוח מקומי
```bash
git clone https://github.com/ethanmimoun/ta35-stocks.git
cd ta35-stocks
# פתח index.html בדפדפן
```
